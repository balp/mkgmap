Main programs

This package contains all the main entry points. For making a map, you need to use {@link uk.me.parabola.mkgmap.main.MakeMap}.

There are other programs for testing. For all these programs you can supply a latitude,longitude on the command line to re-locate the map to somewhere convenient.

MakeTestMap
Creates a very simple map with a few roads on it. This was used to veryfy the basic structure of the file.
MakeTestPointMap
Makes a map that consists only of points. Each point has a different type and subtype. The name of the point is that type and subtype. This way you can investigate what they all mean.
MakeTestPolygonMap
Just like the test point map, but this time for polygons. There should be one for lines too, but there isn't just yet.
